# AI 사기 메시지 방패 (가제)

의심스러운 문자나 채팅을 붙여넣으면 위험도·위험신호·쉬운 경고를 알려주는 AI 사기 탐지 서비스입니다.

> 아직 문제가 확정되지 않았습니다. DAY1~2에서 팀이 함께 문제를 정합니다.

---

## 폴더 구조 안내

```
scam-shield/            ← 저장소 루트
├── docs/               ← 산출물 문서 (.md 파일 모음)
│   └── 00-folder-structure.md   ← 파일 명명 규칙
├── src/                ← 코드 (DAY5부터 생깁니다)
├── tests/              ← 평가셋 (DAY7부터 생깁니다)
├── JOURNEY.md          ← DAY별 진행 일지
├── CHARTER.md          ← 팀 헌장 (역할·브랜치·커밋 약속)
└── README.md           ← 이 파일
```

자세한 파일 명명 규칙은 `docs/00-folder-structure.md`를 확인하세요.

## 팀

| 역할 | 이름 | 한 줄 소개 |
|------|------|-----------|
| PM / 기획 (팀장) | 이도윤 | 문제를 한 문장으로 못 줄이면 시작도 안 한다 |
| 개발 | 박서준 | 일단 작동하는 V1부터, 부끄러워도 내놓는다 |
| 평가 / 데이터 | 최유나 | 숫자로 증명 안 되면 '좋아졌다'는 착각 |
| UX / 검증 | 정하린 | 사용자가 3초 안에 못 알아보면 실패 |
