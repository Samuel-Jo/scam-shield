# AI 사기 메시지 방패

의심스러운 문자나 채팅 내용을 붙여넣으면 위험도·위험 신호·쉬운 경고·행동 안내를 3초 안에 알려주는 사기 탐지 도구입니다.

---

## 해결하는 문제

스미싱·보이스피싱 문자를 받은 사용자가 "눌러도 되나?"를 혼자 판단하지 못해 30초 이상 고민하거나 피해를 입는 문제를 해결합니다. 전문 지식 없이도 3초 안에 위험 여부와 이유, 다음 행동을 확인할 수 있습니다.

---

## 데모 / 접속

### 🌐 웹앱 (바로 체험) — **https://scam-shield-9uc.pages.dev**

설치 없이 브라우저에서 의심 문자를 붙여넣고 분석해 볼 수 있습니다. (예시 10개 버튼 내장,
위험도·위험 신호·추천 행동·PII 마스킹까지 동일 엔진으로 동작) · 소스: `app/index.html`

### 💻 CLI (터미널)

이 저장소는 Node.js만 있으면 바로 실행할 수 있습니다. 별도 설치·빌드가 필요 없습니다.

```bash
# 특정 문자 분석
node src/cli.mjs "분석할 메시지"

# 예시 — 스미싱 탐지
node src/cli.mjs "[국세청] 미납 세금 확인 http://bit.ly/x 지금 본인인증"

# 예시 — 정상 문자
node src/cli.mjs "내일 2시 강남역에서 직거래 가능할까요?"

# 인수 없이 실행하면 내장 예시 3개(스미싱·선입금·정상)를 자동 시연
node src/cli.mjs

# 평가셋 20개 자동 채점 (Pass Rate 측정)
node src/evalRunner.mjs
```

웹앱이 https://scam-shield-9uc.pages.dev 에 배포되어 있습니다(`app/index.html`). 커스텀 도메인·HTTPS는 Cloudflare Pages 기본 제공.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 위험도 점수 | 0~100점, 안전·주의·위험·매우위험 4단계 등급 |
| 위험 신호 | 탐지된 신호 유형과 해당 문구를 목록으로 표시 |
| 쉬운 경고 | 전문용어 없이 누구나 이해할 수 있는 한 줄 경고 문장 |
| 행동 안내 | "링크를 누르지 마세요", "금융감독원(1332) 또는 경찰청(112)에 신고하세요" 등 다음 행동 안내 |
| 가드레일 4종 | PII 마스킹(전화·계좌번호 가운데 `****` 치환), 입력 검사, 출력 검증, 프롬프트 인젝션 탐지 |

## 실행 방법

**필요한 것**: Node.js (버전 제한 없음, LTS 권장). 별도 패키지 설치 불필요.

```bash
# 1. 저장소를 내 PC로 복제
git clone https://github.com/Samuel-Jo/scam-shield.git
cd scam-shield

# 2. 바로 실행
node src/cli.mjs "의심되는 문자를 여기에 붙여넣으세요"
```

## 기술 스택

| 영역 | 내용 |
|------|------|
| 탐지 엔진 | 규칙 기반 V2 — 7종 신호 + 고위험 티어 보너스 + 콤보 보너스 5종 |
| 실행 환경 | Node.js CLI (빌드 불필요, 즉시 실행) |
| 지식 베이스 | RAG식 키워드 KB(`highRiskPatterns`) — 신고 사례 기반, 지속 확장 가능 |
| 가드레일 | PII 마스킹·입력 검사·출력 검증·인젝션 탐지 4종 (`src/guardrails.mjs`) |
| 성능 | Pass Rate 90%(18/20, DAY 13 실측) |
