# 문제 후보 B — 학교 공지 쉬운말 변환

## 누구의 문제인가 (대상 사용자)

- 대학교 신입생 (1학년 입학 첫 학기, 학사 용어가 낯선 상태).
- 대학에 재학 중인 외국인 유학생 (한국어를 어느 정도 읽을 수 있지만, 행정 문어체가 어려운 경우).
- 처음 대학 행정 절차를 혼자 처리해야 하는 지방 출신 또는 첫 독립 생활 학생.

## 어떤 상황에서 겪는가

- 학교 포털에 올라온 수강신청 안내 공지를 읽을 때. "수강 정정 기간 내 이수 구분 변경 불가" 같은 문장이 무슨 뜻인지 몰라 절차를 놓칩니다.
- 장학금 신청 공지를 읽다가 "소득분위 산정 기준일 기준 가구원 소득 합산액" 같은 표현에서 멈추고, 결국 신청을 포기하는 상황.
- 학사 경고·재수강 규정 공지를 읽었지만 이해가 안 돼서 학과 사무실에 전화했더니 "공지 잘 읽어보세요"라는 답변만 받은 경험.

## 반복적으로 겪는 페인 포인트 (관찰/인터뷰 근거)

- 정하린이 신입생 오픈채팅방 3곳을 2주간 관찰한 결과, "이 공지 무슨 말이에요?", "이 서류 어디서 받아요?" 같은 질문이 매일 5건 이상 올라왔습니다. 같은 질문이 반복되었고, 선배들도 매번 설명하는 것에 지쳐 "공지 다시 읽어봐"라고 답하는 경우가 늘었습니다.
- 인터뷰한 유학생 A(중국 출신, 한국어 중급)는 "공지를 번역기로 돌려도 행정 단어는 번역이 이상하게 나와서 3번 읽고도 모를 때가 많다"고 했습니다. 마감일을 하루 놓쳐 장학금 신청을 못 한 경험을 이야기했습니다.
- 신입생 B는 "수강신청 전날 밤에 공지를 읽었는데 무슨 의미인지 몰라서 밤새 선배 카톡을 기다렸다"고 했습니다. 결국 원하는 과목을 신청하지 못했습니다.
- 학과 조교를 인터뷰했을 때, "같은 내용을 반복해서 안내하는 데 하루 1~2시간을 쓴다"는 답변이 나왔습니다. 조교 입장에서도 반복 안내가 부담이었습니다.

## 지금은 어떻게 (불편하게) 해결하고 있나

- 신입생 오픈채팅방에 질문을 올리고 선배의 답변을 기다립니다. 답이 늦거나 틀릴 수 있습니다.
- 학과 사무실에 직접 전화하거나 방문합니다. 운영 시간이 제한적이고, 전화를 어려워하는 학생이 많습니다.
- 공지 전체를 번역기(파파고·ChatGPT)에 붙여 넣어 번역합니다. 행정 용어의 문맥 번역이 부정확해 다시 혼란이 생깁니다.
- 모르면 그냥 무시합니다. 중요한 마감을 놓치는 가장 흔한 이유입니다.

## 이 문제가 중요한 이유

- 학사 공지를 이해하지 못해 마감을 놓치면 학점·장학금·졸업 요건에 직접 영향을 미칩니다. 불이익이 실질적입니다.
- 대학 입학 초기에 겪는 혼란은 학업 적응 전체에 영향을 주어, 중도 이탈 위험을 높인다는 연구 결과가 있습니다.
- 같은 질문이 반복되기 때문에 "공지 → 쉬운 말 변환" 자동화만 해도 학생과 조교 양쪽의 부담이 줄어듭니다.
- NLP의 텍스트 변환·요약 기능으로 접근 가능한 문제입니다. 행정 문어체를 쉬운 구어체로 바꾸는 것은 언어 모델이 잘 하는 영역입니다.
